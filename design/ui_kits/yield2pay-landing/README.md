# Yield2Pay Landing — UI kit

High-fidelity recreation of the Yield2Pay marketing landing page (the "Material / metal"
art direction), composed from the system's own primitives: `Button`, `Eyebrow`,
`StatPanel`, `MetalCard`, `IconTile`, `Badge`.

- **`index.html`** — interactive recreation. Header with EN/PT language toggle, hero with the
  floating investment `StatPanel`, the three-step "How it works" grid, and the final CTA slab.

This is a cosmetic recreation for design reference, not production code. The original lives in
the source project as `Yield2Pay.dc.html`.
