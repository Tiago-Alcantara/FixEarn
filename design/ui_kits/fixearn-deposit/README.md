# FixEarn Deposit — UI kit

The capital-deposit / onboarding flow: choose an amount, pick which tools the returns
should cover, and confirm. Composed from `Input`, `Select`, `SegmentedControl`, `Switch`,
`MetalCard`, `ProgressBar`, `Button`, `Badge`, `Divider`.

- **`index.html`** — interactive 3-step flow (Deposit → Tools → Confirm) with a live returns
  projection that updates as you change the amount.

Cosmetic recreation for design reference; the projection math is illustrative.
