# FixEarn Dashboard — UI kit

Logged-in product view: the company's capital at work, returns, and the subscriptions
those returns are paying for. Composed from system primitives (`StatPanel`, `StatTile`,
`MetalCard`, `SubscriptionRow`, `ProgressBar`, `Button`, `Badge`, `Divider`).

- **`index.html`** — Overview screen: app bar, capital + returns stat tiles, the investment
  `StatPanel`, and a "Subscriptions paid by returns" panel with coverage.

Cosmetic recreation for design reference. Localization (PT-BR) follows the same string map as
the landing kit; this screen is shown in EN for brevity.
